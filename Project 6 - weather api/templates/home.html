Weather Data API

URL Format for specific temperature at specific date: http://127.0.0.1:5000/api/v1/station/date

URL example: http://127.0.0.1:5000/api/v1/10/1988-10-25

URL Format for all dates: http://127.0.0.1:5000/api/v1/station

URL example: http://127.0.0.1:5000/api/v1/10

URL Format for a particular year: http://127.0.0.1:5000/api/v1/yearly/station/year

URL example: http://127.0.0.1:5000/api/v1/yearly/10/1990
